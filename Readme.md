# Rest API with Flask

A Sample Project to showcase the Implimentation of Rest API

## Getting Started

Install Python in Your System

### Prerequisites

Install Flask using pip

### Installing

Fireup your Terminal and hit these Commands
1. pip install flask

## Running the tests

Hit this command in your Terminal
1. python manage.py
